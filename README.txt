Разработка игры Виселица на Python
Цель
Разработать консольную игру "Виселица" на Python с
использованием слов из текстового файла для закрепления
навыков работы с файлами и логикой игрового процесса

Функции проекта
Загрузка слов – случайный выбор слова из текстового файла.

Угадывание букв – игрок вводит буквы, программа проверяет их наличие в слове.

Подсчёт попыток – ограниченное число ошибок (обычно 6-8).

Визуализация – отображение угаданных букв (например, я_локо), оставшихся попыток.

Проверка ввода – обработка некорректных символов и повторных букв.

Повтор игры – предложение сыграть снова после победы или поражения.
